#### 政策选择困境

```
情景: 外储告急 (30) + 民心尚可 (60)

选项A: IMF援助
  → 外储 +25 (75)
  → 民心 -10 (50) ⚠️ 进入危险区

选项B: 出售资产
  → 外储 +20 (50)
  → 民心 -7 (53) ⚠️ 仍然危险

选项C: 鼓舞演讲
  → 民心 +12 (72) ✅
  → 但外储不动...

选项D: 什么都不做
  → 投机者攻击 → 民心继续下降
```

**结论**: 玩家无论怎么选都会顾此失彼

```tsx
function CrisisWarning({ state }) {
  const warnings = [];
  
  if (state.public_support < 50) {
    warnings.push({ 
      level: 'danger', 
      icon: '⚠️', 
      text: '民心告急',
      detail: '建议使用鼓舞演讲或民生补贴'
    });
  } else if (state.public_support < 60) {
    warnings.push({
      level: 'warning',
      icon: '📊',
      text: '民心下降',
      detail: '注意维护社会稳定'
    });
  }
  
  if (state.public_support < 45) {
    warnings.push({
      level: 'critical',
      icon: '🚨',
      text: '危险区域',
      detail: '投机者将发动恐慌攻击！'
    });
  }
  
  return (
    <div className="crisis-warnings">
      {warnings.map(w => (
        <div key={w.text} className={`warning ${w.level}`}>
          {w.icon} {w.text}: {w.detail}
        </div>
      ))}
    </div>
  );
}
```
